Домашнее задание к уроку "JavaScript: типы данных, операторы, циклы"

1. Сумма чисел. Предложить пользователю ввести два числа и вывести их сумму

2. Генератор случайных чисел

а) Имитация бросания двух игральных костей (выпадают числа от 1 до 6)

б) Имитация игры "орёл-решка"

в) Создать скрипт, принимающий целое число от пользователя, а затем выводящий случайное целое число от 1 до принятого числа

г) Создать скрипт, принимающий два числа от пользователя, а затем выводящий случайное число в диапазоне принятых чисел

3. (работа с кавычками) Используя конструкцию if..else, написать скрипт спрашивающий "Каково "официальное" название JavaScript?".
Если пользователь введёт "ECMAScript", вывести "Верно!", иначе вывести "Не знаете? "ECMAScript"!"
4. (оператор '?') Перепишите if с использованием оператора '?':
if (a + b меньше 4)
result = 'Мало'
else
result = 'Много'
